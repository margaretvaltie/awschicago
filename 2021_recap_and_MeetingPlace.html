Moving to MeetingPlace

We've decided to move the group from Meetup to MeetingPlace.io! This event platform lets you control the number of event emails, has a better user experience, allows non-HTML email (with attachments!), and lets you save the dates to multiple calendar tools. About the AWS Chicago user group

This AWS user group is a user-run, user-focused community in Chicago. We hold monthly, free events open to all levels of AWS users. We aim for consistent format with user-focused, quality non-sales presentations and discussion.
