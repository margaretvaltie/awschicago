About the User Group

The Chicago AWS group has been around since 2009 and has grown to 5,000 members. All events are free and open to all. We welcome AWS users from all backgrounds and any experience level.
In-person events are located at different host offices in the Loop during the week with pizza/drinks around 5pm, 2 to 3 quality non-sales presentations and then wrap with q&a before 8pm.

Virtual events are either during the workday for short discussions or in the evening for longer presentations. All virtual events require sign-up and are hosted on Zoom.
Note: We are not run by AWS, but frequently work with Amazon SAs and account reps. As a local community group we resepect our members' privacy and will never distribute personal details.

Please read our Code of Conduct. By attending an AWS Chicago user group, you agree to comply with the group's code of conduct.

About the AWS Chicago user group

This AWS user group is a user-run, user-focused community in Chicago. We hold monthly, free events open to all levels of AWS users. We aim for consistent format with user-focused, quality non-sales presentations and discussion.
